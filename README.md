## Diagrama UML

### Classes e Interfaces

#### Interface ReprodutorMusical
- Métodos:
  - `tocar()`
  - `pausar()`
  - `selecionarMusica(String musica)`

#### Interface AparelhoTelefonico
- Métodos:
  - `ligar(String numero)`
  - `atender()`
  - `iniciarCorreioVoz()`

#### Interface NavegadorInternet
- Métodos:
  - `exibirPagina(String url)`
  - `adicionarNovaAba()`
  - `atualizarPagina()`

#### Classe iPhone
- Implementa:
  - `ReprodutorMusical`
  - `AparelhoTelefonico`
  - `NavegadorInternet`
